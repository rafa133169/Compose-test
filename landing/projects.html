Proyectos

FluxSYS – Gestión Inteligente de Inventarios

Sistema que automatiza el control de inventario, genera reportes en tiempo real y previene desabastecimientos. Utiliza cifrado de datos, autenticación por roles y conexiones HTTPS para máxima seguridad.

Sistema de Información para la Localidad de Cancún

App web que integra APIs de clima, transporte y turismo para ofrecer información en tiempo real a turistas y residentes. Desarrollada con React, Node.js, Express y MySQL.

Sistema de Gestión de Pedidos · Glass Caribe

Aplicación web para administración de pedidos, clientes y proveedores. Incorpora extracción de datos XML, gráficas en tiempo real, WebSockets y panel administrativo. Desarrollado con React, MySQL, Express y Node.js.

E-Commerce Deportivo

Tienda online con catálogo de productos de marcas reconocidas (Nike, Puma, Adidas), gestión de inventario y pagos vía PayPal y Mercado Pago. Dashboard administrativo para seguimiento de pedidos y ventas.

Sistema de Gestión para Taller Mecánico

Sistema CRUD seguro para organización de tareas, personal y clientes. Incluye cifrado de datos, autenticación multifactor, conexión HTTPS y gestión de accesos por roles. Seguimiento del estado de trabajos en tiempo real.
